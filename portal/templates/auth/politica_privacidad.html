Política de Privacidad y Protección de Datos Personales

Centro de Atención Psicológica Online — CAP Online

Última actualización: Junio 2026
Sistema: CAP Online, plataforma académica para atención psicológica en línea.

La presente Política de Privacidad tiene como objetivo informar a los usuarios de CAP Online sobre la forma en que se recopilan, utilizan, almacenan y protegen sus datos personales dentro de la plataforma. Esta política se elabora con base en principios de licitud, consentimiento, información, calidad, finalidad, lealtad, proporcionalidad y responsabilidad en el tratamiento de datos personales.

CAP Online es un sistema web orientado a la gestión de pacientes, especialistas, citas, comunicación mediante chat, solicitudes de videollamada y registro de notas clínicas. Debido a la naturaleza de la información tratada, se aplican medidas de seguridad para proteger la confidencialidad, integridad y disponibilidad de los datos.

Al registrarse en CAP Online, el usuario declara haber leído esta Política de Privacidad y acepta que sus datos sean tratados únicamente para el funcionamiento del sistema y la prestación de los servicios disponibles dentro de la plataforma.

1. Identidad y finalidad del sistema

CAP Online es una plataforma desarrollada con fines académicos para simular y gestionar procesos relacionados con la atención psicológica en línea. Su finalidad principal es permitir que pacientes y doctores puedan interactuar de manera organizada mediante perfiles, citas, mensajes y notas clínicas controladas.

2. Datos personales que se recopilan

Para el funcionamiento de la plataforma, CAP Online puede recopilar los siguientes datos:

  • Nombre completo del usuario.
  • Correo electrónico utilizado para inicio de sesión y recuperación de contraseña.
  • Número telefónico, cuando el usuario lo proporcione.
  • Contraseña de acceso, almacenada de forma cifrada mediante Django.
  • Datos de perfil del paciente o doctor.
  • Especialidad, cédula, experiencia y biografía en el caso de doctores.
  • Historial de citas solicitadas, aprobadas, rechazadas o finalizadas.
  • Mensajes enviados dentro del chat interno de la plataforma.
  • Solicitudes de llamada o videollamada.
  • Notas clínicas registradas por el doctor autorizado durante o después de la atención.

3. Datos sensibles

Se consideran datos sensibles aquellos relacionados con el estado emocional, observaciones clínicas, recomendaciones psicológicas, historial de atención y cualquier información que pueda revelar aspectos de la salud mental del paciente.

Estos datos serán tratados con mayor cuidado y solo estarán disponibles para el doctor correspondiente y para las funciones estrictamente necesarias dentro del sistema. No deberán ser compartidos con usuarios no autorizados.

4. Finalidades del tratamiento de datos

Los datos personales serán utilizados para las siguientes finalidades:

  • Crear y administrar cuentas de usuario.
  • Permitir el inicio y cierre de sesión de forma segura.
  • Diferenciar los roles de paciente, doctor y administrador.
  • Gestionar citas psicológicas dentro del sistema.
  • Permitir comunicación mediante chat interno.
  • Gestionar solicitudes de llamada o videollamada.
  • Registrar notas clínicas por parte del doctor autorizado.
  • Mostrar información de especialistas disponibles.
  • Enviar mensajes relacionados con recuperación de contraseña o funcionamiento del sistema.
  • Mantener trazabilidad básica de actividades relevantes dentro de la plataforma.

5. Uso limitado de la información

La información proporcionada por los usuarios no será utilizada para fines comerciales, publicitarios, venta de datos, análisis externo ni cualquier otro propósito ajeno al funcionamiento de CAP Online. El tratamiento de los datos se limita a las funciones necesarias para la operación del sistema.

6. Medidas de seguridad implementadas

CAP Online incorpora medidas técnicas y administrativas para proteger la información de los usuarios, entre ellas:

  • Autenticación mediante correo electrónico y contraseña.
  • Cifrado de contraseñas utilizando el sistema de autenticación de Django.
  • Protección CSRF en formularios para evitar solicitudes no autorizadas.
  • Validación de datos ingresados por los usuarios.
  • Control de acceso basado en roles.
  • Restricción de notas clínicas únicamente para doctores autorizados.
  • Separación de vistas y permisos entre paciente, doctor y administrador.
  • Uso de sesiones seguras para controlar el acceso al sistema.
  • Evitar consultas SQL directas mediante el ORM de Django.

7. Control de acceso por roles

El sistema maneja diferentes niveles de acceso para proteger la información:

  • Paciente: puede gestionar su perfil, consultar especialistas, solicitar citas, enviar mensajes y acceder a sus funciones permitidas.
  • Doctor: puede consultar información necesaria de sus pacientes, atender citas, responder mensajes y registrar notas clínicas.
  • Administrador: puede gestionar usuarios, doctores y configuraciones generales del sistema.

8. Confidencialidad de la información

La información personal, clínica y de comunicación será tratada de manera confidencial. Los datos no serán divulgados a terceros ajenos al sistema, salvo que exista autorización expresa del usuario o una obligación legal aplicable.

9. Conservación de los datos

Los datos se conservarán mientras la cuenta del usuario permanezca activa o mientras sean necesarios para cumplir con las finalidades del sistema. En caso de que el usuario solicite la eliminación de su cuenta, se revisará la procedencia de dicha solicitud conforme a las necesidades operativas, académicas y de seguridad del sistema.

10. Derechos del usuario

El usuario podrá solicitar el ejercicio de sus derechos de acceso, rectificación, cancelación u oposición respecto de sus datos personales. Estos derechos permiten:

  • Acceso: conocer qué datos personales se tienen registrados.
  • Rectificación: solicitar la corrección de datos incorrectos o desactualizados.
  • Cancelación: solicitar la eliminación de datos cuando sea procedente.
  • Oposición: oponerse al tratamiento de datos para finalidades específicas.

11. Responsabilidades del usuario

El usuario se compromete a:

  • Proporcionar información verídica y actualizada.
  • No compartir su contraseña con otras personas.
  • Cerrar sesión al terminar de utilizar el sistema.
  • No utilizar cuentas de otros usuarios.
  • Reportar cualquier uso no autorizado de su cuenta.
  • Usar la plataforma únicamente para los fines establecidos.

12. Transferencia de datos

CAP Online no transfiere datos personales a terceros con fines comerciales. En caso de que en el futuro se requiera compartir información con algún proveedor tecnológico para el funcionamiento del sistema, se deberá garantizar que dicho proveedor mantenga medidas de seguridad y confidencialidad adecuadas.

13. Consentimiento del usuario

El consentimiento se obtiene cuando el usuario marca la casilla de aceptación durante el registro. Si el usuario no acepta esta Política de Privacidad, no podrá completar la creación de su cuenta dentro de CAP Online.

14. Cambios a la política

CAP Online podrá modificar esta Política de Privacidad cuando sea necesario para mejorar la seguridad, actualizar funcionalidades, cumplir requisitos académicos o adaptar el sistema a nuevas prácticas de protección de datos. Cualquier cambio relevante deberá ser informado dentro de la plataforma.

15. Contacto

Para dudas, aclaraciones o solicitudes relacionadas con el tratamiento de datos personales dentro de CAP Online, el usuario podrá comunicarse con el administrador del sistema o con el responsable del proyecto académico.
